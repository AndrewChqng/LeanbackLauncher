# LeanbackLauncher
仿照Android TV 7.1 Google LeanbackLauncher设计的TV桌面

需要在Android 7.1源码下编译使用

#### 支持功能如下
(1)添加，删除收藏应用

(2)应用启动，移动，卸载

(3)网络状态，时间更新

(4)语音，文字搜索（需要GMS包）
